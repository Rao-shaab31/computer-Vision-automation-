# computer-Vision-automation-
This project uses OpenCV and Python to detect colors in real time via camera. It identifies colors like red, green, and blue, draws boxes around them, and labels them—demonstrating computer vision for automation tasks like robotics and sorting systems.
⚙️ Features
Real-time color detection
Detects multiple colors simultaneously
Draws bounding boxes around objects
Displays object area and FPS
Automatic camera detection
🛠️ Technologies Used
Python
OpenCV
NumPy
📂 How It Works
Detects available camera
Captures live video frames
Converts frame to HSV color space
Applies color masks
Removes noise using filters
Detects contours (objects)
Draws rectangle and labels detected color
